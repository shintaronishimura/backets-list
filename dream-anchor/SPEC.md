# DreamAnchor 仕様書

## 概要

DreamAnchorは、バケットリスト（やりたいことリスト）を管理するデスクトップアプリケーション。
夢や目標を登録し、達成時に過去の自分からのメッセージを受け取る体験を提供する。

## 技術スタック

| 層 | 技術 |
|---|---|
| フロントエンド | React + TypeScript (Vite) |
| バックエンド | Rust (Tauri) |
| データ保存 | JSONファイル（`dreams.json`）+ ローカルgit管理 |
| デスクトップ基盤 | Tauri v2 |

## 機能仕様

### 1. アイテム登録

- タイトル（やりたいこと）を入力して夢を登録する
- カテゴリを以下の3種類から選択する
  - 旅・冒険
  - スキル・成長
  - ライフスタイル・体験
- 未来の自分へのメッセージを任意で入力する
- 「夢を繋ぎ止める」ボタンで登録を実行する

### 2. アイテム一覧表示

- 登録されたアイテムを一覧表示する
- 各アイテムにはタイトルとカテゴリを表示する
- ステータスが `active` のアイテムにはエイジング効果が適用される（後述）
- ステータスが `completed` のアイテムは緑のボーダーと透明度で表示される

### 3. エイジング効果

- `last_touched_at` からの経過日数に応じてアイテムがグレースケール化する
- 計算式: `grayscale = 100 - (経過日数 × 3.33)%`
- 約30日間タッチしないと完全に白黒になる
- アイテムをクリックすると `last_touched_at` が更新され、色が戻る（Touchアクション）

### 4. 達成機能

- `active` アイテムの「達成！」ボタンを押すとステータスが `completed` に変わる
- 達成時に登録時に書いた「未来の自分へのメッセージ」がモーダルでポップアップ表示される
- モーダルはクリックまたは「閉じる」ボタンで閉じられる

### 5. Touchアクション

- `active` アイテムのコンテンツ部分をクリックすると `last_touched_at` が現在時刻に更新される
- エイジングをリセットして色を復元する操作

## データモデル

### BucketItem

| フィールド | 型 | 説明 |
|---|---|---|
| `id` | String (UUID v4) | 一意識別子 |
| `title` | String | やりたいことのタイトル |
| `category` | String | カテゴリ |
| `status` | String | `"active"` または `"completed"` |
| `created_at` | DateTime\<Utc\> | 作成日時 |
| `last_touched_at` | DateTime\<Utc\> | 最終タッチ日時 |
| `future_message` | String | 未来の自分へのメッセージ |
| `photos` | Vec\<String\> | 写真パスのリスト（将来拡張用） |

## データ永続化

- データは `dreams.json` にJSON形式で保存する
- 保存先: OSのアプリデータディレクトリ内の `dreams_repo/` フォルダ
  - Windows: `C:\Users\[ユーザー名]\AppData\Roaming\[アプリ名]\dreams_repo\`
- `dreams_repo/` はローカルgitリポジトリとして初期化され、データ変更のたびに自動コミットされる
  - コミットメッセージ例: `Add: エベレスト登山`、`Update Status: 〇〇 to completed`、`Touch: 〇〇`

## バックエンドAPI（Tauriコマンド）

| コマンド | 引数 | 説明 |
|---|---|---|
| `get_items` | なし | 全アイテムを取得する |
| `add_item` | `title`, `category`, `future_message` | アイテムを追加する |
| `update_item_status` | `id`, `status` | ステータスを更新する |
| `touch_item` | `id` | `last_touched_at` を現在時刻に更新する |

## UI仕様

- 最大幅800px、中央寄せのレイアウト
- カラー
  - アクティブアイテムのボーダー: `#2563eb`（青）
  - 達成済みアイテムのボーダー: `#10b981`（緑）
  - 背景: `#f8fafc`
- モーダルはスケールインアニメーション付きで表示される
